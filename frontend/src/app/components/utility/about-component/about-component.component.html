movie O aplikaciji

Ova aplikacija omogućava upravljanje bioskopima, salama, filmovima i rezervacijama. Kreirana je kao projekat za prikaz rada sa Angular frontend tehnologijom i Spring Boot backend servisom.

Korisnici mogu pregledati, dodavati, menjati i brisati podatke o bioskopima, salama, filmovima i rezervacijama, a sistem koristi REST API komunikaciju između klijenta i servera.

Cilj projekta je demonstracija kompletne CRUD funkcionalnosti i upotreba Angular Material dizajna.
